# clouddomain-trabajo

Landing page for Facebook ads targeting Mexico. Promotes [Cloud Domain Limited](https://order.withintheclouddomain.com/) online work platform.

## Deploy

Deploy to [Netlify](https://www.netlify.com/) with publish directory `landing-mx` (see `netlify.toml`).

Suggested site name: `clouddomain-trabajo`

## Structure

```
landing-mx/
  index.html    Main landing page (es-MX)
  style.css     Styles
  script.js     Meta Pixel click tracking
  privacy.html  Privacy policy
  terms.html    Terms of service
  about.html    About page
```
